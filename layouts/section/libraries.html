KiCad libraries are community contributed and available for download on GitHub. If you wish to contribute to the libraries, refer to the contribution guide.

Latest Libraries

The latest complete set of official KiCad libraries are available as Git repositories at the following links. Each library set can also be downloaded as a compressed archive, for users who do not wish to use Git.

{{ partial "github_lib_table.html" . }}

Users who wish to keep their libraries up to date with the latest additions should clone the library repositories using Git. Using Git means that only the changes to the library need to be retrieved, and significantly reduces download bandwidth.

Stable Library Releases

The libraries are periodically compiled into stable releases which can be downloaded from the links below.

Currently schematic symbols are grouped in libraries which are sets of two files, one named with *.lib* extension and another with *.dcm*. Libraries are generally named in such a way that it describes the symbols it stores, in other words, by category. KiCad Library Editor should be used to edit symbols, however, alternatively, files can be edited using a text editor because they are human readable. The libraries can be found at https://github.com/KiCad/kicad-library[kicad-library] repository. On the other hand, footprints are placed in multiple repositories which have their names suffixed with *.pretty*. Similarly to symbol libraries, repository names indicate the footprints category. Inside the repository, each footprint has its own file whose name ends up with *.kicad_mod* extension. Like symbol library files, *.kicad_mod* files are human readable and can be edited manually, nevertheless it is recommended to use PCB footprint editor to edit/re-save files.
