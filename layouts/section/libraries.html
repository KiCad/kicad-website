KiCad libraries are community contributed and available for download on GitHub. If you wish to contribute to the libraries, refer to the contribution guide.

Latest Libraries

The latest complete set of official KiCad libraries are available as Git repositories at the following links. Each library set can also be downloaded as a compressed archive, for users who do not wish to use Git.

{{ partial "github_lib_table.html" . }}

Users who wish to keep their libraries up to date with the latest additions should clone the library repositories using Git. Using Git means that only the changes to the library need to be retrieved, and significantly reduces download bandwidth.
