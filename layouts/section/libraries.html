KiCad libraries are community contributed and available for download on GitHub. If you wish to contribute to the libraries, refer to the contribution guide.

Library License

KiCad library license information can be found here.

Library Organization

The libraries corresponding to v4.xx of KiCad software are organised on GitHub as follows:

Symbols, 3D models and project templates are stored together in the kicad-library repository.

Footprint libraries are stored as individual repositories with the `.pretty` extension.

Future of the libraries

In conjunction with the KiCad v5 release, the libraries will be reorganized into four separate repositories:

  • kicad-symbols
  • kicad-footprints
  • kicad-packages3d
  • kicad-templates
This will significantly improve user access and reduce the current contribution complexities.

Updating Library Data

Users who wish to keep their libraries up to date with the latest additions should clone the library repositories using Git. Using Git means that only the changes to the library need to be retrieved, and significantly reduces download bandwidth.
