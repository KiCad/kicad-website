Body dimensions
Prefix Field Description Notes
B or Body
  • Body size (`X`x`Y`x`Z`)
  • Height parameter `Z` may be omitted
  • Prefifx `B` may be omitted if the body size is the major dimension used to define the footprint
P or PitchPad pitch
  • Pitch (distance) between pins, pads or leads
  • e.g. P0.63 - 0.63mm pitch between pads
D or DiaDiameter
  • Diameter of major axis for cylindrical component
  • e.g. D10 - 10mm diameter
LLength
  • Length along major axis of component
  • Body size B should be used in preference, where possible
  • e.g. L15.3 - 15mm length
WWidth
  • Width along minor axis of component
  • Body size B should be used in preference, where possible
  • e.g. W4.75 - 4.75mm width
HHeight
  • Height of component measured from PCB
  • Body size B should be used in preference, where possible
  • e.g. H1.1 - component measures 1.1mm above PCB
TThicknessComponent thickness, where appropriate
LDLead dimensions Used if speficic lead dimensions need to be called out
PadPad dimensions
  • Only required for non-standard pads, or where multiple options are available
  • e.g. PAD2.25x1.75 - pad size is 2.25mm x 1.75mm
TabTab dimensions
  • Dimension of tab (either mechanical or soldering)
  • e.g. T3x1.5 - tab size is 3.0 x 1.5mm
EPExposed pad
  • Only used if there is one more more exposed pads under a component
  • If there is more than one exposed pad, prepend with the pad count
  • If exposed pad dimensions need to be added, append them here e.g. EP2.4x3.6 - 2.4mm x 3.6mm
CLClearance Clearance distance, e.g. between rows of pads
